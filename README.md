# TakaraTone

Download page and user manual for **TakaraTone** — a free, local desktop app that
builds playlists for Tidal, Spotify, and YouTube Music, with chart time-travel,
live concerts, and artist news.

- **Download:** see the [Releases](../../releases) page (or the site below).
- **Manual:** [TakaraTone_Manual.html](TakaraTone_Manual.html)
- **Site:** https://golgi-ap.github.io/takaratone-app/

This repository hosts only the public download page, the manual, and the installer
releases. The application source is kept in a separate private repository.
